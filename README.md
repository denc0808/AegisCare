---
📖 项目简介

AegisCare（安护智伴）是一款专为老年群体打造的本地化AI健康服务应用，聚焦居家老人健康监护、慢病咨询、用药提醒、情感陪伴等核心需求，采用极简大字体护眼界面，降低老年用户操作门槛。

项目基于大模型+RAG检索增强技术搭建，内置专业老年健康知识库，支持本地私有化部署，保障用户健康数据隐私安全，打造低成本、易操作、全天候的老年智能健康守护解决方案。


---
✨ 核心功能

👴 老年用户专属功能

- AI健康问诊：内置慢病、养生、用药知识库，精准解答老人健康疑问

- 定时用药提醒：自定义用药时间，智能提醒，避免漏服错服

- 极简护眼界面：大字体、高对比度、宽松布局，适配老人视觉习惯

- 温情陪伴聊天：AI智能陪聊，缓解老人孤独，提供心理慰藉

- 健康科普推送：日常养生、慢病护理、饮食作息知识科普

🔧 技术核心特性

- 本地私有化部署，数据不上云，保障隐私安全

- RAG向量检索，精准匹配健康知识，回答更专业

- 无复杂操作，老人一键使用，家属可远程辅助

- 轻量化运行，低配设备也可流畅启动

🚀 快速启动

1. 环境依赖安装

# 克隆项目到本地
git clone git@github.com:denc0808/AegisCare.git
cd AegisCare

# 安装项目依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

2. 启动项目

python app.py

3. 访问使用

浏览器打开：http://127.0.0.1:5000，即可进入AegisCare长者健康AI平台
